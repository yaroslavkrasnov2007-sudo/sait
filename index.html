Что такое движок?

Программное обеспечение, обеспечивающее базовые технологии для создания видеоигр: графику, звук, физику и скриптинг. Предоставляет абстракцию над низкоуровневым кодом.

Актуальность

В отличие от перегруженных гигантов (Unreal/Unity), Cube Engine идеален для изучения работы систем рендеринга и сборки небольших 3D-приложений без излишнего overhead'а.

АРХИТЕКТУРА ЯДРА

Схема Фреймворка

Замкнутый цикл (Game Loop) объединяет модули ввода, рендеринга и скриптов в единую отказоустойчивую систему.

while(!window.Closed()) {
  UpdatePhysics();
  RenderFrame();
}

Ядро (C++)

Максимальное быстродействие и прямое управление памятью. Стандарт AAA-индустрии.

Подсистемы

Single-Window Flow, Менеджер событий (Callbacks), Асинхронное логирование.

VS 2022

Инструментарий

Visual Studio 2022, Git, Win32 API для глубокой интеграции с ОС Windows.

ГРАФИЧЕСКИЙ КОНВЕЙЕР

DirectX 11 API

Баланс мощности и контроля. Поддержка HLSL 5.0 для расчета сложного освещения и PBR материалов.

Vertex Shader: Расчет Model-View-Projection.
Pixel Shader: Свет, текстуры, карты нормалей.

Управление Ассетами

3D Модели (OBJ/FBX)

Автоматический парсинг геометрии и генерация буферов (VBO/IBO) для быстрой отрисовки на GPU.

Сериализация сцены

Запись позиций, поворотов и компонентов всех объектов сцены в JSON/XML для сохранения прогресса.

UI / UX Интерфейс

Визуальная среда для сборки уровней. Включает окно сцены, иерархию объектов и инспектор свойств.

C# Скриптинг

Внедрена интеграция с C# (Mono). Позволяет писать высокоуровневую логику поведения объектов без рекомпиляции ядра C++.

Рабочее пространство
